# 트리

  n개의 노드를 가진 트리는 n-1개의 간선을 가진다.

1) 트리의 구현

  1. 이진 트리가 아닌 트리는 그래프를 이용하여 구현한다.

  2. 이진 트리의 구현

     - 일차원 배열을 이용한 구현
    
       완전 이진트리나 포화 이진트리 구현 시 사용 because 다른 트리는 공간의 낭비가 심해짐

       부모가 i일 때 자식은 2*i , 2*i+1

     - 이중 배열을 이용한 구현
  
        node[0]이 root노드일 때 node[0][1]가 left, node[0][2]가 right이다.    

     - 구조체를 이용한 구현
